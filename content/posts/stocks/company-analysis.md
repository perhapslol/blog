# 上市公司分析方法

## 基本面分析

基本面分析是评估公司内在价值的方法，主要关注：

1. **财务报表分析** - 利润表、资产负债表、现金流量表
2. **行业分析** - 市场份额、竞争格局、行业趋势
3. **管理层评估** - 战略规划、执行能力、诚信记录

## 估值方法

- 市盈率法
- 市净率法
- 现金流折现法

## 关键指标

```
市盈率 = 股价 / 每股收益
市净率 = 股价 / 每股净资产
ROE = 净利润 / 净资产
```
