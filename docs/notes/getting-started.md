# Getting Started

Welcome to the digital garden. This is a place where ideas grow and evolve.

## What is This?

This is a digital garden - a collection of notes, thoughts, and projects that are constantly evolving. Unlike a traditional blog with linear articles, a digital garden allows for connections between ideas and continuous updates.

## Features

- **Full-text Search**: Use the search bar to find any content on the site
- **Comments**: Leave comments using GitHub OAuth
- **Tag System**: Organize content with tags and categories
- **Dark Mode**: Switch between light and dark themes

## How to Use

1. Browse the navigation to explore different sections
2. Use the search bar to find specific content
3. Leave comments on articles using your GitHub account
4. Enjoy the responsive design on any device
