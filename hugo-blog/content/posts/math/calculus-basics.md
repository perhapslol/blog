## 导数

导数表示函数的变化率。

### 定义
```
f'(x) = lim(h→0) [f(x+h) - f(x)] / h
```

### 常见导数
- (x^n)' = n*x^(n-1)
- (sin x)' = cos x
- (e^x)' = e^x

## 积分

积分是导数的逆运算。

### 不定积分
```
∫f(x)dx = F(x) + C
```

### 定积分
```
∫[a,b] f(x)dx = F(b) - F(a)
```
