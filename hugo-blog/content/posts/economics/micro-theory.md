## 供需理论

微观经济学研究个体经济行为和市场机制。

### 需求定律

需求定律指出，在其他条件不变的情况下，商品价格上升，需求量减少；价格下降，需求量增加。

**需求曲线公式：**
$$Q_d = a - bP$$

其中：
- $Q_d$ = 需求量
- $P$ = 价格
- $a$ = 截距
- $b$ = 斜率

### 供给定律

供给定律指出，在其他条件不变的情况下，商品价格上升，供给量增加；价格下降，供给量减少。

**供给曲线公式：**
$$Q_s = c + dP$$

其中：
- $Q_s$ = 供给量
- $P$ = 价格
- $c$ = 截距
- $d$ = 斜率

### 市场均衡

市场均衡发生在需求量等于供给量的时候：
$$Q_d = Q_s$$

解这个方程可以得到均衡价格和均衡数量。

## 弹性理论

### 需求价格弹性
$$E_d = \frac{\%\Delta Q_d}{\%\Delta P}$$

### 供给价格弹性
$$E_s = \frac{\%\Delta Q_s}{\%\Delta P}$$

弹性的大小决定了价格变化对需求量或供给量的影响程度。
